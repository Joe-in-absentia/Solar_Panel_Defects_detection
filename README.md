## 📌 About

AI-powered Solar Panel Defect Detection system built using deep learning. The model classifies solar panel images into multiple defect categories with confidence scores.

This project compares **MobileNetV2, ResNet50, and EfficientNetB0**, and selects the best-performing model for deployment using Streamlit.

---

## 🚀 Features

* 📤 Upload solar panel image
* 🔍 Predict defect type
* 📊 Show prediction confidence
* 🖱️ Simple and interactive interface

---

## 🧠 Model Details

* **Models:** MobileNetV2, ResNet50, EfficientNetB0
* **Type:** Transfer Learning
* **Framework:** TensorFlow / Keras
* **Input Size:** 224 × 224

### Classes:

* Bird-drop
* Clean
* Dusty
* Electrical-damage
* Physical-Damage
* Snow-Covered

---

## 📊 Model Performance

* MobileNetV2 Accuracy: ~81%
* ResNet50 Accuracy: ~88% ✅
* EfficientNetB0 Accuracy: ~83%

## 🏁 Conclusion

This project shows how deep learning can automate solar panel defect detection. Among the tested models, **EfficientNetB0 performed best**, providing strong accuracy and efficient performance for real-world use.

The Streamlit interface makes the system interactive and user-friendly, showcasing a complete end-to-end AI solution with real-world relevance.
